Tartami - Product Overview
Tartami is a Somali‑rooted, premium, approval‑based auction platform designed for trust, privacy, and cultural authenticity.

Core Principles
All bidders must be approved before bidding
All items are consigned at the item level, not auction level
Auctions follow increment‑only bidding
Soft‑close with anti‑sniping
Buyer premium:
  - 10% regular
  - 15% special auctions
Payments happen offline (cash, transfer, mobile money)
Admins mark invoices as paid
Settlement generates consignor payouts
Identity masking is always respected

User Roles
Bidder — approved participant
Consignor — item owner
Admin — full control, but bounded by rules

High‑Level Flows
User registers → admin approves
User submits item → admin approves
Auction goes live → users bid
Auction ends → invoices generated
Admin marks payments → settlement
Consignors receive payouts
